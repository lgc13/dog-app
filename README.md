# dog-app (api example)

1. Create react app

2. Install axios

```bash
yarn add axios
```

3. Import axios

```bash
import axios from 'axios'
```

4. You can now use axios requests as per here:

https://github.com/axios/axios

Examples:

```js
axios.get('https://someUrl.com').then(response => console.log(response.data))
```
